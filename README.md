## Features

Dynamic Target
A floating target that gains speed and unpredictability as your streak increases.

Persistent Environment
Successful hits leave permanent faint Red X's, while misses leave solid bullet holes on the background.

Momentum-Based Movement
Smooth, physics-based movement ensuring the target is always gliding and never stops.

Progress Saving
High scores and lifetime points are automatically saved to your local machine.

Real-time Stats
Instant feedback on reaction time in milliseconds, current streak, and total points.

## Installation

1. Download .NET SDK: https://dotnet.microsoft.com/en-us/download
2. git clone https://github.com/q1lra/reaction-trainer
3. cd path/to/reaction-trainer-master
4. dotnet run
